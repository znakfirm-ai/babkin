# TZ_001: Нижняя навигация (Bottom Nav)

## Контекст
Проект: Telegram Mini App, фронтенд: Vite + React + TypeScript. Экран HomeScreen уже подключен через App.tsx.

## Цель
Добавить нижнюю навигацию из 5 пунктов: Главная, Обзор, Добавить, Отчёты, Настройки.

## Требования
- Навигация всегда видна снизу экрана.
- Нижнее меню статично и всегда видно на экране, кроме состояний ввода (когда открыта клавиатура, выбор даты, редактирование полей и другие полноэкранные элементы ввода), где оно может временно скрываться.
- 5 кнопок одинаковой ширины.
- Активный пункт визуально отличается (достаточно текста/простого индикатора).
- Нажатие переключает текущий экран (пока без роутера, можно через local state в App).
- “Добавить” пока ведёт на пустой экран-заглушку “Добавить”.

## Экранные заглушки
Создать экраны в `frontend/src/screens/`:
- HomeScreen (уже есть)
- OverviewScreen (текст “Обзор”)
- AddScreen (текст “Добавить”)
- ReportsScreen (текст “Отчёты”)
- SettingsScreen (текст “Настройки”)

## Критерии приёмки
- При клике на пункты меню меняется экран и подсвечивается активный пункт.
- Меню не перекрывает контент экрана (есть отступ снизу).
- `npm run dev` работает без ошибок.
