# AI Knowledge Base

Эта папка содержит документацию агентов и skills,
используемых для разработки проекта babkin.

## Важно
- Это документация.
- Не используется в runtime.
- Не влияет на сборку frontend/backend.
- Не импортируется в код.

## Структура
- agents/ — роли и подходы к работе
- skills/ — практики и паттерны

Используется как база знаний при разработке.
